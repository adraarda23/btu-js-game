# BTU JS Game

## Oyun Hakkında

## Nasıl Başlatılır?

1. Proje dosyalarını bilgisayarınıza klonlayın veya indirin.
2. Tarayıcınızda `index.html` dosyasını açın.
3. Oyunu oynamaya başlayın!
